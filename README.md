# YouTube Summarizer with Gemini AI

A web app that uses Google's Gemini 2.0 Flash to summarize any YouTube video.

## Tech Stack
- Python Flask
- Google Gemini AI (Vertex AI)
- Google Cloud Run

## Live Demo
https://youtube-summarizer-1061426331395.us-central1.run.app

## Usage
Paste a YouTube link and get an AI-generated summary instantly.
Best results with videos under 5 minutes.

## Future Improvements
- Support for longer videos
- Summary download option
- Multiple language support
